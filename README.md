wishare-cli
===========

## Install

```bash

$ npm i -g wishare-cli

```

## Usage
update via patch
================
```bash

$ wishare patch

```
update via minor
================
```bash

$ wishare minor

```
update via major
================
```bash

$ wishare major

```

rollback version.Note:must have the version updated
================
```bash

$ wishare version 0.0.2

```
