wishare-cli
===========

### Introduce
wishare,一款命令行工具,用于winFED
### Lists
- [admin-panel](./admin-panel.md) 用于生成一款基于angular.js、bootstrap的后台框架


## Install

```bash

$ npm i -g wishare-cli

```

## Usage
update via patch
```bash

$ wishare patch

```
update via minor
```bash

$ wishare minor

```
update via major
```bash

$ wishare major

```

rollback version.Note:must have the version updated
```bash

$ wishare version 0.0.2

```
